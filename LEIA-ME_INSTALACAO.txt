# GUIA DE INSTALAÇÃO - ANALISADOR DE MARCHA

### O que você precisa ter instalado:
1. **Python 3.11, 3.12 ou 3.13** (IMPORTANTE: NÃO use Python 3.14!)
   - Baixe em: https://www.python.org/downloads/
   - ⚠️ **VERSÕES COMPATÍVEIS**: Python 3.11, 3.12 ou 3.13
   - ⚠️ **VERSÃO INCOMPATÍVEL**: Python 3.14 (não funciona!)
   - ⚠️ **IMPORTANTE**: Marque a opção "Add Python to PATH" durante a instalação!
   - ⚠️ **IMPORTANTE**: Marque a opção "Add Python tkinter" durante a instalação!

2. **Git** (opcional, mas recomendado)
   - Baixe em: https://git-scm.com/download/win
   - Use as configurações padrão durante a instalação

---

## Como instalar e usar:

### INSTALAÇÃO AUTOMÁTICA (RECOMENDADO)

1. **Dê duplo clique** em: `INSTALAR_E_CRIAR_EXE.bat`

2. O script irá automaticamente:
   - ✓ Verificar se Python está instalado (versão 3.11-3.13)
   - ✓ Instalar todas as bibliotecas necessárias:
     • numpy, opencv-python, matplotlib, Pillow, tkinter
     • tensorflow, jax, warp-lang, mujoco-mjx
     • monocular-demos
   - ✓ Criar um atalho "Analisador de Marcha.lnk" na pasta do projeto

3. **Aguarde** (pode levar 5-10 minutos na primeira vez)

4. Pronto! Um atalho será criado na pasta do projeto!

---

## Como usar o programa:

1. Após a instalação, você verá o arquivo **"Analisador de Marcha.lnk"** na pasta do projeto

2. Para executar o programa:
   - **Dê duplo clique** no atalho "Analisador de Marcha.lnk"
   - OU execute manualmente: abra o terminal na pasta e digite `python video_processor_gui.py`

3. **IMPORTANTE**: 
   - Python DEVE estar instalado para o programa funcionar
   - As bibliotecas são instaladas no seu sistema Python
   - Os projetos ficam salvos na pasta "projetos"
   - Não mova os arquivos desta pasta

---

## Problemas Comuns:

### "Python não encontrado" ou "Versão incompatível"
- Instale Python 3.11, 3.12 ou 3.13: https://www.python.org/downloads/
- **NÃO instale Python 3.14** (incompatível com as bibliotecas)
- Marque "Add Python to PATH" durante a instalação
- Reinicie o computador após instalar

### "Git não encontrado" 
- Opção 1: Instale o Git: https://git-scm.com/download/win
- Opção 2: O script tentará instalar sem Git (pode ser mais lento)

### "Erro ao instalar bibliotecas"
- Verifique sua conexão com a internet
- Execute o script como Administrador (clique direito → Executar como administrador)

### "Atalho não funciona"
- Verifique se Python está no PATH do sistema
- Execute `python --version` no terminal para confirmar
- Execute o programa diretamente: `python video_processor_gui.py`

## Para outras pessoas usarem:

Para que outra pessoa use o programa:

2. Copie toda a pasta do projeto
3. Na máquina dela, execute `INSTALAR_E_CRIAR_EXE.bat` para instalar as bibliotecas

## Suporte:

Se encontrar problemas:
1. Execute `DIAGNOSTICO.bat` para verificar o que está instalado
2. Verifique se Python está instalado e é versão 3.11-3.13: 
   - Abra o Prompt de Comando e digite `python --version`
   - Se for Python 3.14, desinstale e instale Python 3.11 ou 3.12
3. Execute o script `CORRIGIR_INSTALACAO.bat` como Administrador
4. Se ainda houver erro, execute `INSTALAR_E_CRIAR_EXE.bat` como Administrador
5. Leia as mensagens de erro na tela preta
6. Certifique-se de ter conexão com internet
